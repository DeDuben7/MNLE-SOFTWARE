Filters incoming MIDI messages.

This file filters incoming MIDI messages in MIDI functions. When filterd it will send the messages to the corresponding function.

uint8_t MIDI_PROC ( uint8_t  MIDI_MSG)

Function for processing MIDI messages.

Parameters
MIDI_MSGis an eight bit integer wich is a byte of the incoming MIDI message @val None
Returns
iError

Checks if incoming message is first or second byte. If first byte, it will check wich function and post a corresponding flag. If second byte, checks wich flag is posted and sends the data array to corresponding function.
